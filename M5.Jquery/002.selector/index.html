Jquery selector là gì?

CSS selector là bộ chọn các thẻ html mà các bạn mong muốn ví dụ : .class , #id , tag_name (body , div , p ....)

Jquery là bộ chọn cuả jquery để các thẻ html mà các bạn mong muốn

    css => jquery
    .class => $(".class")
    #id => $("#id")
    tg_name (div) => $("tag_name") $("div")
    .classA .classB => $(".classA .classB")
    .classA >.classB => $(".classA >.classB")
    chú ý tất cả các select tên class , tên id , tên thẻ đều phải nằm "" hay ''
    nhưng mà document và window thì không cần trong "" hay ''

    $(document).ready();là gì
    $() là cú pháp của select jquery
    document là đại diện của toàn trang html
    ready có nghĩa là khi maf các trang web đc tải song
    và chúng sẽ viết các lện jquery hc js ở trong method ready
    để đảm bảo js đc cahyj sau khi trang web đã đc tải xuống

    tiến hành trừ tiền trong tài khoản
    hành động trừ tiền sẽ diễn ra sau cuộc gọi
    và nó là callback của cuộc gọi hay tin nhán đó

    function() {
        // code thực thi bên trong callback
    }
